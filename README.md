# Geek-O-Metre

Geek-O-Metre is a simple web-based quiz application created with HTML, CSS and JavaScript.

The quiz includes questions about science, technology, books, fantasy worlds, games and popular culture.

## Live Demo

Add the GitHub Pages link here after publishing the project:

`https://your-username.github.io/geek-o-metre/`

## Features

- 10 quiz questions
- 4 answer choices for every question
- Questions stored in a JavaScript array
- Random question order
- Random answer order
- 15-second timer for every question
- Correct and wrong answer feedback
- Short explanation after every answer
- Progress bar
- Live score display
- Different result levels
- Restart button
- Responsive design for desktop and mobile devices
- The next button stays disabled until the question is answered
- If the time ends, the correct answer is shown automatically

## Technologies

- HTML5
- CSS3
- JavaScript
